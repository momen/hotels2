Configuration of project
1.  Download the files from repository and place on server.
2. Make sure your server meeting the prerequisite as described in https://laravel.com/docs/5.4/installation
3. Update the composer.
4. Go to the folder location config/app.php , set the project url to the variable "presenturl".

Assumptions
1. To search the Hotel Name, Destination - you can type the text in search box. But as per the project scope for Price range you need to type [start_value]:[end_value] i.e ex: $100:$200.
2. To search Date range similarly you need to type 10-10-2020:15-10-2020
3. To sort the result with title and price need to click on the header.
